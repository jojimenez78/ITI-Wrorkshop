## Cómo sobrevivir en bash

`pwd`: Muestra el directorio actual.

`ls`: Lista los archivos y carpetas del directorio actual.

`ls -a`: Lista todos los archivos, incluidos los ocultos.

`ls -la`: Lista todos los archivos con detalles (permisos, tamaño, fecha).

`cd <directorio>`: Cambia al directorio especificado.

`cd ..`: Sube un nivel en la jerarquía de carpetas.

`mkdir <nombre>`: Crea un nuevo directorio.

`rm <archivo>`: Elimina un archivo.

`rm -r <directorio>`: Elimina un directorio y todo su contenido.

`cp <origen> <destino>`: Copia un archivo o directorio al destino.

`mv <origen> <destino>`: Mueve o renombra un archivo o directorio.

`touch <archivo>`: Crea un archivo vacío.

`nano <archivo>`: Abre o edita un archivo en el editor Nano.

`cat <archivo>`: Muestra el contenido de un archivo.

`explorer .`: Abre el explorador de Windows en el directorio actual.

`code .`: Abre el directorio actual en Visual Studio Code.
